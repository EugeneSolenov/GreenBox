# Проект:"GreenBox" (Доставка здоровой еды)
**Практическое занятие 7: Работа с CSS-фреймворками**

## Описание
Адаптивный лендинг пейдж для сервиса доставки правильного питания.
Дизайн выполнен в минималистичном стиле с использованием ярких цветовых акцентов и полноэкранных изображений.

## Технологический стек
*   **HTML5** — семантическая разметка.
*   **CSS Framework: Bulma (v1.0.0)** — использован для сетки (Grid), навигации, кнопок и форм.
*   **Custom CSS** — дополнительные стили для:
    *   Реализации фонового изображения с затемнением (Overlay).
    *   Создания ярких "Vibrant" кнопок с тенями.
    *   Плавной прокрутки (`scroll-behavior: smooth`).
*   **FontAwesome** — векторные иконки.
*   **JavaScript** — минимальный скрипт для работы мобильного меню (бургер).

## Особенности реализации
1.  **Hero-секция:** Вместо стандартного цвета использовано фоновое изображение с CSS-градиентом (`linear-gradient`) поверх него. Это обеспечивает читаемость белого текста на любой фотографии.
2.  **Цветовая схема:**
    *   Основной: `#10ac84` (Зеленый)
    *   Акцентный (CTA): `#0984e3` (Яркий синий)
    *   Текст: `#2d3436` (Темно-серый для контраста)
3.  **Адаптивность:** Сайт корректно отображается на мобильных устройствах, меню сворачивается в "гамбургер", карточки выстраиваются в одну колонку.

*   `index.html` — основной файл с версткой и стилями.

# Проект: Многостраничный сайт "GreenBox"

## Описание проекта
Проект расширен с одностраничного лендинга до полноценного многостраничного веб-сайта сервиса доставки здорового питания.

**Тема:** Сервис доставки еды "GreenBox".
**Фреймворк:** Bulma CSS (v1.0.0).

## Структура проекта
В проекте реализовано 4 html-страницы:

1.  `index.html` — **Главная страница**. Содержит приветственный экран (Hero), преимущества сервиса и призыв к действию.
2.  `contacts.html` — **Контакты**. Включает:
    *   Форму обратной связи (стилизованные поля input, select, textarea, checkbox).
    *   Контактную информацию.
    *   Блок с картой (iframe).
3.  `faq.html` — **FAQ (Частые вопросы)**. Реализована с использованием тегов `<details>` и `<summary>`, стилизованных под аккордеон Bulma.
4.  `404.html` — **Страница ошибки**. Креативная страница, на которую попадает пользователь при переходе по несуществующей ссылке. Содержит кнопку возврата на главную.

## Технологии и решения
*   **HTML5/CSS3**: Семантическая верстка.
*   **Bulma CSS**: Использование классов фреймворка для сетки (Columns), карточек (Cards), форм, кнопок и типографики.
*   **FontAwesome**: Иконки для визуального оформления.
*   **JavaScript**: Скрипт для работы мобильного меню ("бургер") подключен ко всем страницам.
*   **Навигация**: Реализована подсветка активного пункта меню в зависимости от открытой страницы.
