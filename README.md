# Проект:"GreenBox" (Доставка здоровой еды)
**Практическое занятие 7: Работа с CSS-фреймворками**

## Описание
Адаптивный лендинг пейдж для сервиса доставки правильного питания.
Дизайн выполнен в минималистичном стиле с использованием ярких цветовых акцентов и полноэкранных изображений.

## Технологический стек
*   **HTML5** — семантическая разметка.
*   **CSS Framework: Bulma (v1.0.0)** — использован для сетки (Grid), навигации, кнопок и форм.
*   **Custom CSS** — дополнительные стили для:
    *   Реализации фонового изображения с затемнением (Overlay).
    *   Создания ярких "Vibrant" кнопок с тенями.
    *   Плавной прокрутки (`scroll-behavior: smooth`).
*   **FontAwesome** — векторные иконки.
*   **JavaScript** — минимальный скрипт для работы мобильного меню (бургер).

## Особенности реализации
1.  **Hero-секция:** Вместо стандартного цвета использовано фоновое изображение с CSS-градиентом (`linear-gradient`) поверх него. Это обеспечивает читаемость белого текста на любой фотографии.
2.  **Цветовая схема:**
    *   Основной: `#10ac84` (Зеленый)
    *   Акцентный (CTA): `#0984e3` (Яркий синий)
    *   Текст: `#2d3436` (Темно-серый для контраста)
3.  **Адаптивность:** Сайт корректно отображается на мобильных устройствах, меню сворачивается в "гамбургер", карточки выстраиваются в одну колонку.

## Структура проекта
*   `index.html` — основной файл с версткой и стилями.
*   `README.md` — документация проекта.

# Проект: GreenBox (Многостраничный сайт)
**Практическое занятие 8**

## Описание
Проект расширен до многостраничного сайта на базе CSS-фреймворка **Bulma**. 
Реализована навигация между страницами, единый стиль оформления и страница ошибки 404.

## Структура проекта
*   `index.html` — Главная страница (Лендинг).
*   `contacts.html` — Страница контактов с формой обратной связи и картой.
*   `faq.html` — Страница часто задаваемых вопросов (реализована через HTML тег `<details>` стилизованный под аккордеон Bulma).
*   `404.html` — Кастомная страница ошибки.

## Особенности реализации
1.  **Навигация:** Меню обновлено, добавлена подсветка активной страницы (класс `is-active-page`).
2.  **Форма:** Использованы классы Bulma (`input`, `select`, `textarea`, `checkbox`) для стилизации полей ввода.
3.  **Единообразие:** Хедер и Футер идентичны на всех основных страницах.
4.  **Адаптивность:** Бургер-меню работает на всех страницах (использован JS скрипт).
